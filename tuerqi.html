VPS is a blank server: You get an IP, root access, and Linux. That's it. For AI workloads, you must handle Docker setup, Nginx reverse proxy, SSL, firewall rules, SSH keys, API key management, monitoring, and backups yourself. Budget 5-10 hours/month for maintenance.
Security alert: Over 42,000 AI agent instances have been found publicly accessible without authentication. Many store API keys in plain text. Always use encrypted secret management, restrict network access, and never expose management ports.

Cost Optimization Tips:
- Commit to Savings Plans (AWS), CUDs (GCP), or RIs (Azure) for 1-3yr terms: 30-69% off.
- Platforms like BetterClaw offer Heartbeat scheduling: 60%+ savings on AI API costs.
- Hetzner dominates EU pricing at roughly half the cost of US providers.

GPU Cloud Alternatives: For AI training (not just inference), consider dedicated GPU cloud: Lambda Labs, CoreWeave, RunPod, Vast.ai, or hyperscaler GPU instances (AWS P5, Azure ND H100, GCP A3). Pricing ranges from $1-8/hr per GPU depending on tier.

Useful tools for self-hosted AI:
- Portainer: Docker management UI (no terminal needed)
- Caddy: Simpler alternative to Nginx, auto-SSL
- Tailscale: Secure private networking between VPS nodes
- UptimeRobot: Free monitoring, 50 monitors
- Ansible: Automate server provisioning
Self-hosted vs Managed: Self-hosting gives full control and lower costs at scale. Managed platforms give zero maintenance and built-in security. Break-even is typically at 50+ daily active users or $100+/mo in API spend. Below that, managed is usually cheaper when you factor in your time.
