Совет дня: Выбор между Serverless и VPS

Serverless-функции оптимальны для коротких задач (вебхуки, API, крон). VPS — для долгоживущих процессов (базы данных, ML-инференс, боты). По цене за ресурс Serverless обходится в ~30 раз дороже VPS, но для эпизодических нагрузок общие затраты могут быть ниже (citation:8). Для n8n и автоматизации рекомендуется VPS с 2-4 vCPU, 4-8 ГБ RAM и NVMe SSD (citation:11).

Часто задаваемые вопросы

Какой VPS выбрать для самохостинга LLM?
Для моделей 7B — GPU 16+ ГБ VRAM (RTX 4090). Для 13-70B — A100/H100. Начните с Hetzner GPU или Selectel. Квантизация GGUF позволяет запускать 7B на CPU с 16 ГБ RAM.
Облака безопасны для чувствительных данных?
Крупные провайдеры применяют продвинутое шифрование и обновляют политику кибербезопасности (citation:10). Однако не подходят для объектов, не допускающих онлайн-доступ (электростанции, оборона). В РФ — требования 152-ФЗ.
Serverless vs VPS — что дешевле?
Serverless дороже за ресурс (в ~30 раз), но дешевле для эпизодических задач. VPS — для долгоживущих процессов. При постоянной нагрузке VPS всегда дешевле (citation:8).
Какой провайдер для российского проекта?
Yandex Cloud (полный сервис), VK Cloud (удобство), Selectel (выделенные/GPU), SberCloud (корпоративный), Timeweb (доступный). Оплата в рублях без иностранных карт.
Спотовые инстансы для ИИ — стоит ли?
Да, экономия 50%+. Подходят для обучения с чекпоинтингом и batch-инференса. Не подходят для latency-critical продакшена. Обучение — spot, инференс — on-demand (citation:9).
Какие ИИ-модели использовать в 2026?
API: GPT-4o, Claude 4, Gemini 2.5. Самохостинг: Llama 4, Qwen 3.5, Mistral Large 2. Для русского: YandexGPT 4, GigaChat Max. Для ограниченных ресурсов: Llama 3 8B / Qwen 7B (квантизованные). Alibaba развивает агентный ИИ Qwen (citation:2).
